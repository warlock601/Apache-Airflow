# Astronomer
Astronomer is a managed platform for Apache Airflow. Astro generally runs Airflow within a Docker container.

- Initialize Astro Porject
```bash
astro dev init
```
Whole project strcuture will be created like this: </br>
<img width="363" height="351" alt="image" src="https://github.com/user-attachments/assets/cdf3c8aa-0d00-41c9-b451-0c8705ad7844" />
- To start, basically it will start building the Docker image
```bash
astro dev start
```
